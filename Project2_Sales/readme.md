# Sales Data Analysis Dashboard

## 📌 Objective

To analyze sales data and create an interactive dashboard using Excel.

## 🛠 Tools Used

* Microsoft Excel
* Pivot Tables
* Charts

## 📊 Key Insights

* Total revenue analysis
* Top selling products
* Payment method usage
* Revenue by referral sources

## 📈 Conclusion

This project demonstrates how Excel can be used to analyze business data and create dashboards.
