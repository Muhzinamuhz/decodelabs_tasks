# SQL Data Analysis on E-commerce Dataset

## 📌 Objective

To analyze an e-commerce dataset using SQL and extract meaningful insights.

## 🛠 Tools Used

* MySQL
* SQL
* Excel

## 📊 Key Insights

* Total revenue from all orders
* Total number of orders
* Revenue by referral source
* Average order value
* Orders by payment method
* Most sold products
* Top 5 customers by spending

## 📈 Conclusion

This project demonstrates how SQL can be used to analyze business data and extract valuable insights.
